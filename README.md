# E-Learning Product Owner and System Design Learning Project

## Purpose

This repository is a learning project for practicing Product Owner work, software design, and system design using an e-learning website case study.

The project is not intended to be a real commercial launch. It is designed to help a beginner Product Owner learn how to move from an idea to clear documentation that a software team can understand before implementation.

## Project Idea

The product is an Arabic-first e-learning website for Egyptian secondary school students.

Students can browse teacher courses, redeem prepaid codes, buy courses using student balance, watch recorded lessons, solve MCQ quizzes, and track progress.

Teachers and education centers can publish courses based on the official Egyptian curriculum. Admins manage approvals, users, prepaid codes, balances, reports, and platform settings.

## Learning Goals

- Practice business analysis and Product Owner thinking.
- Learn how to define MVP scope.
- Learn how to write basic software requirements.
- Learn how to organize user roles, user stories, acceptance criteria, and backlog.
- Learn how to prepare documents before software implementation.
- Use the same project later to practice software design and system design.

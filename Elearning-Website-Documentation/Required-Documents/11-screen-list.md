# E-Learning Website - Screen List

## 1. Purpose

This document lists the main screens needed for the MVP.

It helps UI/UX and frontend teams understand what pages should exist.

## 2. Public Screens

| Screen | User | Purpose | Main Actions |
| --- | --- | --- | --- |
| Home Page | Guest | Introduce platform and show course entry points | View courses, register, log in |
| Login | All users | Access account | Enter credentials |
| Register | Student/Parent | Create account | Submit registration form |
| Course Catalog | Guest/Student/Parent | Browse available courses | Search, filter, open course |
| Course Details | Guest/Student/Parent | View course information | View price, teacher, lessons, enroll |

## 3. Student Screens

| Screen | Purpose | Main Actions |
| --- | --- | --- |
| Student Dashboard | Show student learning summary | View enrolled courses, balance, progress |
| Redeem Code | Add value to student balance | Enter prepaid code |
| My Courses | Show enrolled courses | Open course |
| Lesson Player | Watch course lessons | Play video, mark lesson complete |
| Quiz Page | Solve MCQ quiz | Select answers, submit quiz, view score |
| Progress Page | View learning progress | View completed lessons, quiz progress |

## 4. Parent Screens

| Screen | Purpose | Main Actions |
| --- | --- | --- |
| Parent Dashboard | Show linked students | View students and progress |
| Link Student | Connect parent to student | Enter student ID |
| Parent Redeem Code | Add value to linked student balance | Select student, enter code |
| Student Progress View | Monitor student learning | View courses, lessons, quiz progress |

## 5. Teacher Screens

| Screen | Purpose | Main Actions |
| --- | --- | --- |
| Teacher Dashboard | Show teacher summary | View courses, enrollments, sales |
| Teacher Profile | Manage teacher information | Edit name, phone, subject, bio, image, documents |
| Course List | Manage teacher courses | View, create, edit courses |
| Course Editor | Create or edit course | Set year, subject, term, chapter, price, description |
| Lesson Management | Manage course lessons | Add, edit, reorder lessons |
| Quiz Management | Manage MCQ quizzes | Add questions, answers, correct answer |
| Course Students | View enrolled students | View student progress |

## 6. Admin Screens

| Screen | Purpose | Main Actions |
| --- | --- | --- |
| Admin Dashboard | Show platform summary | View users, courses, codes, reports |
| User Management | Manage users | View, update, deactivate users |
| Teacher Approval | Approve teachers | Review teacher data, approve/reject |
| Course Approval | Approve courses | Review course, approve/reject |
| Curriculum Management | Manage curriculum data | Add/edit years, subjects, terms, chapters |
| Prepaid Code Management | Manage prepaid codes | Generate, cancel, track codes |
| Student Balance Management | Manage balances | Add/subtract balance, reset for refund |
| Reports | View platform reports | View codes, enrollments, teacher sales, student progress |
| Settings | Manage basic platform settings | Update simple system settings |

## 7. Notes

- These are MVP screens only.
- Detailed wireframes can be created later from this list.
- Mobile app screens are not included in MVP.
